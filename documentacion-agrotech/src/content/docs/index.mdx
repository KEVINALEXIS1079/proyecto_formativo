## Características Generales

### Autenticación y Autorización
- **JWT Tokens**: Autenticación stateless con refresh tokens
- **Sistema de Permisos**: Control granular basado en roles y permisos directos
- **Guards Automáticos**: Protección automática de endpoints

### Comunicación en Tiempo Real
- **WebSocket**: Actualizaciones en tiempo real para todas las operaciones
- **Namespaces**: Canales separados por módulo
- **Eventos**: Notificaciones automáticas de cambios

### Arquitectura
- **REST API**: Endpoints HTTP RESTful con paginación
- **WebSocket**: Comunicación bidireccional en tiempo real
- **Validación**: DTOs con class-validator para entrada de datos
- **Documentación**: OpenAPI/Swagger integrada

### Tecnologías
- **Backend**: NestJS con TypeScript
- **Base de Datos**: PostgreSQL con TypeORM
- **Cache**: Redis para sesiones y permisos
- **Documentación**: Starlight (Astro)
