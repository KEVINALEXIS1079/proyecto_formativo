## Scripts de Testing

### Crear Usuario de Prueba

```bash
# Crear usuario administrador
curl -X POST http://localhost:4000/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "nombre": "Test",
    "apellido": "User",
    "identificacion": "111111111",
    "correo": "test@agrotech.com",
    "password": "Test123!"
  }'
```

### Crear Lote de Prueba

```bash
# Login primero
TOKEN=$(curl -X POST http://localhost:4000/auth/login \
  -H "Content-Type: application/json" \
  -d '{"correo":"admin@agrotech.com","password":"Admin123!"}' \
  | jq -r '.token')

# Crear lote
curl -X POST http://localhost:4000/geo/lotes \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "nombre": "Lote de Prueba",
    "area": 10000,
    "geometria": {
      "type": "Polygon",
      "coordinates": [[
        [-74.1, 4.6],
        [-74.1, 4.7],
        [-74.2, 4.7],
        [-74.2, 4.6],
        [-74.1, 4.6]
      ]]
    }
  }'
```

Estos datos están diseñados para testing y desarrollo. En producción, asegúrate de usar datos reales y validar todas las entradas.
:::
