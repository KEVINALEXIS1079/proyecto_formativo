Bienvenido a la referencia de la API de Agrotech. Aquí encontrarás detalles sobre los endpoints, modelos de datos y métodos de autenticación.

## Arquitectura de Módulos

### Seguridad y Acceso
- **[Autenticación](/api/auth)**: Gestión de identidades, login y JWT.
- **[Usuarios y Permisos](/api/users)**: Administración de perfiles, roles y control de acceso (RBAC).

### Gestión Agrícola
- **[Mapas y Lotes](/api/geo)**: API Geoespacial para delimitación de terrenos.
- **[Cultivos](/api/cultivos)**: Ciclos de vida, siembra y cosecha.
- **[Actividades](/api/activities)**: Registro de labores culturales y tareas de campo.

### Operaciones y Recursos
- **[Inventario](/api/inventory)**: Control de stock, insumos y maquinaria.
- **[Producción](/api/production)**: Transformación, lotes de producción y ventas.
- **[Finanzas](/api/finance)**: Transacciones, balance y costos operativos.

### Inteligencia y Monitoreo
- **[IoT Central](/api/iot)**: Gestión de dispositivos y telemetría en tiempo real.
- **[Reportes](/api/reports)**: Analítica de datos y reportes gerenciales.
- **[Métricas](/api/iot-reports)**: Análisis detallado de variables ambientales.
